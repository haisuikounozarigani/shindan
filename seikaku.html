Specimen 01 ─ ステータス

称号：さなぎ（成長中） 愛着：恐れ回避型 好み：静けさ 5/5・独り時間 5/5（お気に入りはそばに）

人は好き。お気に入りの人が近くにいると満たされる。

基本パラメータ（Big Five）

強み（パッシブ）

  • 知識の統合 / 抽象思考 / 適応が速い
  • 正確さを最優先（正確さ→スピード→楽しさ→やさしさ）
  • 静かな場所で会話の質が上がる

デバフ / 境界

  • 根拠のない断定 / 高圧 / 強要
  • 求めてないアドバイス / きつい匂い
  • 浅い会話は苦手 / 感情表現・頼るのは修行中

ブースト / リセット

  • 発見・感性・楽しさ → エネルギー↑
  • リセット：猫 ＞ いい匂い ＞ マインドフルネス
  • 集中サイン：脚を組む / 目がゆるむ / 口を触る

好感度が上がったときの挙動（イベント）

  • 勉強以外の話が増える / 甘える / 目で追う / 物理距離が近づく
入口：哲学・研究・感性 くだらない話OK：「あの雲◯◯に見えへん？」 キーワード：変化 / 依存性のある毒 / 草原の大きな石

恋愛スタイル（総合）

はじまり

  • いい匂い、相手の立場で考えられる人、柔軟性に惹かれやすい
  • 一目惚れしやすいが、そこからはゆっくり進む
  • 好きな人なら「近い距離」が好き

コミュニケーション

  • 連絡：仲良くなる前は多め、仲良くなったら少なくてOK
  • 文量は短文 / 通話が楽
  • ひとり時間は5/5必要。ただし、そばにいるだけなら平気

愛着：恐れ回避型

  • 近づきたい＋傷つくのが怖い → ときどき距離を取る
  • 明確な好意表示で安心しやすい
  • ストレス時はいったん置いてから再接近が◎

不安・衝突のハンドリング

  • 不安：まず一人で考える
  • してほしい：好意や誤解の訂正をはっきり伝える
  • 話し合い：考えと目的を説明→納得ラインに調整
  • イライラ時：数分クールダウン後に対話

長続きの鍵 / 安心の交換

  • 鍵：変化・体験の共有・優しさ
  • 贈れる安心：好きになったら基本的に嫌いにならない（謝って笑えたら大体OK）
  • 欲しい安心：明確な好意（言葉/行動で）

相性

  • 開放性が高い / 誠実 / 優しい
  • 極端な現実主義 / 無責任 / 低共感

Do / Don’t

  • 努力をほめる / 気づいてない魅力を言語化
  • 近くに座る・手短な通話・小さな体験を共有
  • 根拠のない断定 / 強要 / 高圧
  • スキンシップ：お気に入りならOK。それ以外はNG

初期デートの当たり

  • 喫茶店でべらべら話す / 動物とふれあう
  • 惚れ直しトリガー：真剣な顔、何かを改善しようとしている姿
